# CMU - Compiler Design
This is the course George Hotz refers to in his talk with Lex Fridman when talking about the compilers course.

[15-411 Compiler Design](https://www.cs.cmu.edu/~janh/courses/411/16/index.html)

The course covers the design and implementation of compiler and runtime systems for high-level languages, and examines the interaction between language design, compiler design, and runtime organization.
Topics covered include:
* Lexical and syntactic analysis.
* Type-checking.
* Program analysis.
* Code generation and optimization.
* Memory management.
* Runtime organization.
